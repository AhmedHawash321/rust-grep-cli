# rust-grep-cli

A professional command-line tool built with **Rust** that searches for a specific string within a file.

## Features
- **Fast Search:** Leverages Rust's zero-cost abstractions for high performance.
- **Case Sensitivity Control:** Switch between case-sensitive and case-insensitive search using environment variables.
- **Robust Error Handling:** Graceful exits with clear error messages for file I/O or input issues.
